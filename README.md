# Corophioid-Phylogenomics-Pipeline

## Comprehensive phylogenomic workflow developed during PhD research.

### Pipeline components:
- Transcriptome assembly
- Quality assessment
- Contamination filtering
- Orthology inference
- Alignment and filtering
- Target-capture marker development
- Target-capture sequence assembly and filtering
- Species tree reconstruction

Tools include Trinity, BUSCO, Evigene, Diamond, OrthoFinder,
PhyloPyPruner, IQ-TREE, PHYLUCE, aTRAM, and custom Bash/Perl scripts.
